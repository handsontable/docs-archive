This page demonstrate example of using Handsontable with Bootstrap.

In the previous releases of Handsontable if you wanted to use Bootstrap classes like table, table-hover, table-striped etc. you had to include plugins/bootstrap/handsontable.bootstrap.css file. Since 0.31.0-beta1 that file is not necessary and all relevant CSS selectors are included to the dist/handsontable.css file.
